Image Search using Unsplash API

  • Go to https://unsplash.com
  • Create an account
  • Click on the "Sign in" button
  • Once logged in click on burger menu > Developers/API
  • Click on "Your Apps" and "New Application"
  • Check all the boxes and agree to terms
  • Give your app a name and click create application
  • Once the app is created in Unsplash developers page move to "Keys" section of the page
  • Note down access key and public key for future usage
  • You can now request to the Unsplash API using
    https://api.unsplash.com/photos/?client_id=YOUR_APPLICATION_ID
    will return images
  • https://api.unsplash.com/search/photos?client_id=YOUR_APPLICATION_ID&query=buildings
    will return images of buildings
  • https://api.unsplash.com/photos/random?client_id=YOUR_APPLICATION_ID
    will return random images
  • There are several other cases that we can use this API. Explore it using this url
    https://www.pluralsight.com/guides/using-the-unsplash-api
